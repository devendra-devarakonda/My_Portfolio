## 🎬 Interactive Experience

### Cinematic Boot Sequence

The portfolio opens with a custom terminal-inspired loading screen that simulates a system initialization process before revealing the main application.

### 3D Hero Scene

The landing section features a live Three.js scene containing:

* Floating animated rings
* Interactive wireframe geometry
* Dynamic lighting effects
* Ambient particle systems
* Mouse-responsive interactions

### Custom Cursor System

A dual-layer magnetic cursor enhances user interaction by adapting to clickable and interactive elements across the application.

### Dynamic Background Engine

A custom particle network runs across the entire site, creating a connected visual experience between sections.
